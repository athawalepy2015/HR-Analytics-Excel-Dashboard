# HR Analytics Excel Dashboard

## Overview
Interactive HR analytics dashboard built in Microsoft Excel to analyze employee attrition trends.

## Features
- KPI cards (Total Employees, Attrition, Attrition Rate, Active Employees, Avg Age)
- Interactive slicers (Gender, Department, Education)
- Job satisfaction rating
- Attrition analysis by:
  - Education
  - Job Role
  - Department
  - Age Group
  - Marital Status

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard formatting
